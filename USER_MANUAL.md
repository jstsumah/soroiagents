## 1. Getting Started

### Logging In

-   **First-Time Login:** If an admin has created an account for you, enter your email address and click "Proceed". You will receive a secure link via email to set your password for the first time.
-   **Returning Users:** Enter your email address, click "Proceed," and then enter your password on the next screen to sign in.
-   **Forgot Password:** If you've forgotten your password, click the "Forgot Password?" link on the password entry screen to receive a reset link via email.

### Signing Up

If you are a new agent, you can register for an account yourself.

1.  From the login page, click the "Sign Up" link.
2.  Enter your full name and email address and click "Proceed."
3.  If your email is not already registered, you will be prompted to enter your company name and create a password.
4.  Upon completion, your account will be created with a "Pending" status. An administrator will be notified to review and approve your account. You will receive an email notification once your account is activated.

## 2. For Agents

### Agent Dashboard

The dashboard is your central hub for important updates and quick access to information. It includes:

-   **Notice Board:** Displays important system-wide announcements from administrators.
-   **Profile Status Alert:** Notifies you if your profile is incomplete or pending approval.
-   **Your Tier & Commission:** Shows your current access tier and the associated commission level.
-   **Quick Stats:** At-a-glance counts of available properties and exclusive deals.
-   **Promotion Card:** Highlights special promotions or news.
-   **Latest Resource:** A direct link to the most recently uploaded resource available to you.
-   **Featured Property:** Showcases a hand-picked property from the portfolio.

### Managing Your Profile

You can view and edit your personal and company information by clicking on your avatar in the top-right corner and selecting "Profile."

-   **Personal Details:** Update your name, phone number, and country.
-   **Company Information:** If you are linked to a company, you can view its details. If your company profile needs updating or has not been created yet, you can click the "Update/Add Company Info" button to manage it. This includes contact info, legal documents, and signed contracts.

### Accessing Resources

The "Resources" section in the sidebar gives you access to all materials based on your assigned tier.

-   **Rates, Deals, Itineraries, etc.:** Each page provides a filtered view of resources specific to that category. You can search and sort to find what you need quickly.
-   **Downloads:** This page consolidates all downloadable files like brochures, factsheets, images, videos, and more. You can filter by category using the tabs at the top.
-   **Downloading:** Simply click the "Download" button on any resource card to get the file.

### Contacting Support

Click the "Contact Support" button at the bottom-right of the screen to open the support chatbot. You can ask questions, and if the bot cannot help, an administrator can join the chat to assist you live.

## 3. For Administrators

### Admin Dashboard

The admin dashboard provides a high-level overview of the entire system, including:

-   **Agent Statistics:** Total, active, pending, and currently online agents.
-   **Agent Tier Distribution:** A pie chart showing the breakdown of agents by tier.
-   **Resource Distribution:** A bar chart showing the number of resources in each category.

### User Management

Accessible via the "Users" link in the sidebar.

-   **Viewing Users:** See a comprehensive list of all users. You can search, filter by type/tier/status, and sort the list.
-   **Adding a User:** Click "Add User" to create a new account manually. You will set a temporary password which you must share with the user.
-   **Editing a User:** From the user list, click the actions menu (...) on any user and select "Edit" to modify their details, including:
    -   **Status:** Activate, deactivate, or mark an account as pending. Activating a user sends them a welcome email.
    -   **Tier & Type:** Change an agent's access level and type (local/international).
    -   **Company Assignment:** Link a user to a company profile.
    -   **Role (Super Admin only):** Promote users to Admin or Super Admin roles.
    -   **"Can View Users" Permission:** Grant a non-admin user read-only access to the user list.
-   **Bulk Actions:** Select multiple users using the checkboxes to change their status or tier in bulk, or to delete them (Super Admin only).

### Company Management

Accessible via the "Companies" link in the sidebar.

-   **Centralized Profiles:** Manage a single, authoritative profile for each company, which can be linked to multiple users. This prevents data duplication.
-   **Adding/Editing:** Create new company profiles or edit existing ones. The system automatically prevents the creation of duplicate companies based on key information like name, phone, and registration numbers.
-   **Document Management:** Upload and manage legal documents and signed contracts for each company.
-   **User Assignment:** From a company's profile page, you can view all linked users and add or remove users from that company.

### Resource Management

Admins can manage all resources from the "Resources" section in the sidebar. Each resource page (Rates, Deals, Itineraries, etc.) includes an "Add" button and an actions menu on each resource card to "Edit" or "Delete" it.

-   **Uploading:** You can upload files directly or link to external URLs.
-   **AI Metadata Suggester:** When adding a new resource, you can paste content into the AI tool to automatically suggest a description and category.
-   **Access Control:** For every resource, you can specify exactly which agent tiers and user types (local/international) can access it.

### Support Chat Management

Navigate to "Support Chat" to view and manage all user chat sessions.

-   **New Sessions:** New, un-actioned chat sessions are highlighted for attention.
-   **Live Chat:** Join any "open" session to chat directly with the user.
--   **Session History:** View the transcript of all "closed" sessions.

### System Settings

The "Settings" page allows Super Admins and Admins to customize the application.

-   **Branding:** Set the login page background, main brand colors, and tier colors.
-   **Company Details:** Update the main contact information for your company, which appears on the agent "Contacts" page.
-   **Email Templates:** Customize the content of automated emails (e.g., User Activation, Password Reset).
-   **Dashboard Cards:** Configure the site-wide "Notice Board" and "Promotion Card."
-   **Integrations:** Add embed codes for third-party tools like the Resova Availability Chart.

### Billing & Audit Trail (Super Admin)

-   **Billing:** Manage the application's subscription plan and view payment history.
-   **Audit Trail:** A critical security feature that provides a read-only log of all significant actions performed in the system, including user management, resource changes, and settings modifications. It records who did what, and when.
