# **App Name**: Tiered Access Hub

## Core Features:

- Tiered Authentication: User authentication and role-based access control (RBAC) restricted to admins and agents; tiered access defined for Bronze, Silver, Gold, and Platinum levels.
- Tiered Resource Display: Displays resources categorized by type and access tier, ensuring agents only see resources relevant to their tier.
- Secure Downloads: Handles secure downloads of resources (rates, itineraries, brochures, images) based on agent tier from Firebase Storage.
- AI-Powered Metadata Suggestion: Uses AI tool to recommend tags, descriptions, and categorizations for uploaded resources to streamline the metadata entry process.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights to add a touch of sophistication.
- Body and headline font: 'Inter' (sans-serif) for a modern and easily readable interface. 
- Use a consistent set of minimalist icons for resource categories to enhance usability and visual appeal.
- Card-based layout for resources to ensure a responsive and intuitive experience on both web and mobile platforms.
- Subtle transition animations when navigating between resource categories to provide a smooth user experience.
